Technologies That I Use

HTML5: The latest version of Hypertext Markup Language, the code that describes web pages. It provides better flexibility and efficiency.

CSS3: The latest version of Cascading Style Sheets, used to style and layout web pages — for example, to alter the font, color, size, and spacing of your content.

JavaScript: A programming language that allows you to implement complex things on web pages. It is the third layer of the layer cake of standard web technologies.

React: A JavaScript library for building user interfaces. It allows developers to create large web applications that can update and render efficiently in response to data changes.
